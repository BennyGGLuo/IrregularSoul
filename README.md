# IrregularSoul

For this semester, I developed a top-down roguelike / survivors-like game as a new project rather than continuing my previous work. My goal was to create a playable vertical slice that demonstrates the core gameplay loop, player progression, enemy pressure, UI flow, and overall presentation of the game in a polished and build-ready state.

Across the three milestones, the project evolved from a basic combat prototype into a more complete playable demo with progression systems, multiple weapons, stage events, win/lose states, persistent upgrades, and improved menu/audio presentation.

This final build represents a vertical slice of the game and is intended to show the direction, gameplay feel, and major interconnected systems of the project.
